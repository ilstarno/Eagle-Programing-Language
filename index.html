Eagle Programing Language - EGP

Eagle Programing Language is an new programing language with a compiler and whole syntax in Albanian Language. This project was as an idea 3 years ago since Albanian Schools do not teach programing to the students. This language has an Basic like syntax.First i thinkd to execute EGP code to Java VM but being albanian language it made really difficult for this, than an i created an indipendent virtual machine and parser with reads eagle programing language code. EGP stands for Eagle Programing Language, and .egp is the file extention of the code container. .exf is stands for eagle executable file. Momentally this project has support only for Windows based OS's since in Albania Windows Operating System is the most used one. Currently we are developing an Web Server wich will make possible to develop web applications. Later we plan to port the compiler and the virtual machine for mobile Os's such as iOS and Android
